# DROPDECK
A Macropad with dual mode.
___________________________________________________________________________________________________________________________________________________________________________

## Overview

Dropdeck is compact custom built macropad with 5 machanical switches,1 infinite rotary encoder , 1 potentiometer and seed studio XIAO RP2040.128X32 pixels display
Dropdeck has 2 modes chill mode for the normal usage like desktop productivity and the Beast mode for the AAA title gaming.
___________________________________________________________________________________________________________________________________________________________________________

## Inspiration

The inspiration for the Dropdeck came from challenge faced by me during the gaming and normal usage to use shortcuts.
The Dropdeck fixes that problems as in chill mode it will help in using shortcuts easily for opening/closing tab,screenshot and in Beastmode or quick game interactions like healing , loadout etc
It is not built to replace the keyboard its built to make life easier while gaming and productive task.
___________________________________________________________________________________________________________________________________________________________________________

## Assembly Guide

• Solder Xiao RP2040 to the PCB

• Solder switches make sure that it exactly passes through the hole

• Solder Display,potentiometer,Rotary encoder

• Now fix PCB to the bottom enclosure with the help of the M2.5 Screws

• Fix the top case with the help of the Screws

## How to use
•  connect it with any USB-C cable

•  The Display shows the mode

• Use switch 5 to change between modes

### chill mode

• switch 1 : Open Tab

• switch 2 : Close Tab

• switch 3 : Screenshot

• switch 4 : Show desktop

• Potentiometer : Analog volume mixer

• Rotary Encoder : Changing volume

### Beast mode

• switch 1 : Heal 

• switch 2 : Tactital ability

• switch 3 : Reload

• switch 4 : Loot/interact

• Potentiometer : Analog volume mixer

• Rotary Encoder : Change inventory
